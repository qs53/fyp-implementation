class SparseCategoricalFocalLoss(tensorflow.python.keras.losses.Loss)
    SparseCategoricalFocalLoss(gamma=2.0, alpha=None, from_logits=False, **kwargs)
 
The class definition for focal loss function to reduce impact of easy instances
on the overall loss.
 
Example:
     ```python
     loss_fn = SparseCategoricalFocalLoss(gamma=2, alpha={0:1,1:3,2:2}, from_logits=True)
     ```
